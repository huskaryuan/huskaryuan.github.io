卡方统计量和卡方分布

若X1,X2,X3,……Xn相互独立, 且都服从 标准正态分布 N(0,1) , 则称随机变量: χ2=X12+X22+X32+……+Xn2服从自由度为n的χ2分布, 记为X2∼χ2

 

下面是卡方分布的实验模拟：

xxxxxxxxxx
from random import sample
from turtle import color
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from pandas import DataFrame, merge, crosstab
from collections import Counter


cates = {'dress': 15, 'shoes': 15, 'sweater': 20, 'jeans': 10, 'shorts': 15, 'pet':5, 'phone':10, 'book': 10}
total = list()
for key, value in cates.items():
    total += [key]*value*1000000


def chi_calc(total):
    sample_total = sample(total, 100000)
    total_dict = dict(Counter(sample_total))
    sub_square = 0
    res = 0
    for k, v in total_dict.items():
        sub_square = (v - cates[k])**2
        res += sub_square/v
    return res


chi_list = list()
chi_list = [chi_calc(total) for _ in range(500)]
# x轴直方图
sns.set()  # 声明使用 Seaborn 样式
plt.hist(chi_list,  facecolor='#94B49F')
plt.title("CHI")

拟合优度检验

检验一个总体概率分布是否服从某个历史或理论的概率分布，即为拟合优度检验。

1. 拟合优度检验步骤

  1. 建立原假设H0和备择假设H1

  2. 抽样并记录观察频数fi

  3. 计算原假设为真的情况下的期望频数ei=理论概率*样本容量

  4. 计算卡方统计量χ2=∑1k(fi−ei)2ei

  5. 拒绝法则（单侧检验）

    • p值法 如果p−≤α,拒绝H0
    • 临界值法 如果χ2≥χα2，拒绝H0

2. 拟合优度检验例子

在某购物网站， 基于过去某段时期内，历史数据统计结果表明，各场景商品转化占比如下

-Ps：搜索转化概率 30% -Pl：列表转化概率 40% -Pr：推荐转化概率 30%

需判断当前用户习惯是否发生改变

  • 一、建立建设

    • 原假设H0：Ps=0.3，Pl=0.4，Pr=0.3，未明显转变
    • 备择假设H1：Ps≠0.3，Pl≠0.4，Pr≠0.3，有明显转变

  • 二、10000 个订单商品的抽样中，3050 个从搜索转化，4050 个从列表转化，2900 个从推荐转化

  • 三、原假设为真的情况下，3000 个从搜索转化，4000 个从列表转化，3000 个从推荐转化。

  • 四、计算卡方统计量χ2=502/3000+502/4000+1002/3000=4.79

  • 五、查表可知，当自由度=2时，p-value 为0.09，不能拒绝原假设，即不能认为用户购物习惯发生了变化

# python 代码实现
from scipy import stats
p = {'s': 0.3, 'l': 0.4, 'r': 0.3}
obs_freq = {'s': 3050, 'l': 4050, 'r': 2900}
sample_size = sum(obs_freq.values())
exp_freq = {key: int(value*sample_size) for key, value in p.items()}
chi_stats = sum( (obs_freq[key]-exp_freq[key])**2/exp_freq[key] for key, value in obs_freq.items())
p_value = 1 - stats.chi2.cdf(chi_stats, 2)
crit = stats.chi2.ppf(0.95, 2)

独立性检验

从一个总体中抽取样本，并记录两个分类变量的观测值

1. 独立性检验步骤

  1. 1建立原假设H0和备择假设H1

  2. 抽样并记录观察频数fij

  3. 计算原假设为真的情况下的期望频数eij=理论概率*样本容量

  4. 计算卡方统计量χ2=∑i∑j(fij−eij)2eij

  5. 拒绝法则（单侧检验）

    p值法 如果p−≤α,拒绝H0

    临界值法 如果χ2≥χα2，拒绝H0

2. 独立性检验例子

需判断不同性别用户在品类偏好上是否有明显差异

  • 一、建立建设

    • 原假设：购物偏好与性别独立
    • 备择假设：购物偏好与性别不独立

  • 二、观察频数fij如上表所示

  • 三、若原假设为真。

  • 四、计算卡方统计量χ2=502/3000+502/4000+1002/3000=6.45
